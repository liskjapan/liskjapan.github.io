あなたの「独自ブロッチェーンを簡単に構築したい」を実現します。
強力なLisk SDKにより、独自ブロッチェーンを簡単に構築できます。
合意アルゴリズム、取引速度、トークン発行数、ノード数などを自由に設定し、自分だけのアルトコイン（トークン）を簡単に作成できます。構築されたアルトコインは、双方向ペグによってLiskのメインチェーン（LSK）と交換することができます。

JavaScript
Liskは世界で最も一般的に使用されているプログラミング言語 "JavaScript" で開発されているブロックチェーン・アプリケーション・プラットフォームです。LiskのSDKはサイドチェーンテクノロジーを利用することで、開発者がカスタムトークンを含む強力でスケーラブルなブロックチェーン・アプリケーションの構築を可能にします。Liskは使いやすさとアクセシビリティに焦点を当て、世界No.1のブロックチェーン・アプリケーション・開発プラットフォームになることを目指しています。Liskはスイスのツークに本社を置き、2016年第1四半期のICO期間中に14,000BTC以上のBitcoinを調達しました。Liskのユーティリティ・トークン（LSK）は、金融庁から正式認可されている大手仮想通貨取引所 "bitFlyer" "coincheck" にも上場されている、非常に安全でクリーンなブロックチェーンです。

Sidechain technology
Lisk SDK（Sidechain Development Kit）を使えば、独自ブロックチェーンを簡単に構築することができます。Liskのツールは全てオープンソースのJavaScriptで書かれており、そして誰でも無料で使用することができます。サイドチェーンで世界に広がる可能性を発見してください。


Application

作成したアプリケーションは、サイドチェーン上の動的な機能にアクセスするためのインターフェイスをすべての人に提供します。サイドチェーンに接続されると、アプリケーションはLiskのメインチェーンとデータを交換できます。

Sidechain

サイドチェーンはLisk SDKを使うことで簡単に構築・カスタマイズできる、個人的なブロックチェーンと考えてください。サイドチェーンはあなたのアプリケーションと連携し拡大しながら、データを格納していきます。

Mainchain

Liskのメインチェーンは現在LSKをホストしており、将来的には全てのサイドチェーンのハブとなります。プラットフォームが稼働すると、サイドチェーンを登録したり、独自ICOを実施できるようになります。

Lisk Hubは預金額の確認や送金の実行、トランザクションの確認、Lisk IDの管理と言った基本機能にプラスして、デリゲートへの投票機能も搭載したオールインワンソリューションとなっています。Lisk Hubは将来的にメインチェーンへの独自トークン作成機能、ICO実施機能、さらにはサイドチェーンの登録機能を実装する予定です。

Use cases of Lisk
Liskのブロックチェーンは汎用性が高く、様々な分野で活用されることを想定しています。ブロックチェーンが持つ「耐改ざん性」「トレーサビリティ」「ゼロダウンタイム」といった特性を活かし、アイデア次第で全く新しいサービスを構築することも可能です。Liskなら、あなたの考えるサービスを実現できると信じています。


1. ゲーム
ゲームユーザー同士で直接レアアイテムの交換したり、アイテムに付加価値をつけて販売する。

2. コンテンツクリエイター
コンテンツを作成・管理・共有するそれぞれの人に対し報酬の分配を行う。

3. サプライチェーン
物品がどこから来て、中身が何なのか、そしてそれがどのようにあなたに届いたのかを正確に知る。

4. 寄付
寄付金がどういったルートで、どう使われたのかを正確に提供する。

5. 医療
プライバシーの問題無く、ボーダレスに医療記録へ効率的にアクセスし保存する。

6. 不動産
詐欺を防止し、公共の不動産記録を使用して正当な所有権を取得する。

7. 自動運転
自動運転車に革新的なサービスと強固なネットワークを提供する。

8. ファイル管理
データの損失を防ぎ、暗号化を使用してファイルのアクセス許可を管理する。

9. 法的文章
複雑な操作をすることなくデータセットの法的完全性を保証する。

10. 人事管理
改ざん防止システムを使用して、教育データや履歴書などの資格情報資産などを管理する。

Lisk CommanderはLisk Core APIのCLIです。将来的には、ブートストラップとサイドチェーンの管理をサポートする予定です。それは低いエントリーポイントと開発者にとって最も便利なツールを目指します。
